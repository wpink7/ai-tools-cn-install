AI 开发工具国内安装指南

国内网络环境下，最简单地安装 Node.js 24、Claude Code、Codex 和 OpenClaw。

Node.js 24 AI CLI 配置样例 镜像链接

1. 安装 Node.js 24

Linux / macOS 用 nvm，Windows 直接下载 Node.js 24 安装包。核心是走国内镜像。

Linux / macOS nvm + 国内 Node 镜像 
git clone https://gitee.com/mirrors/nvm.git ~/.nvm
cd ~/.nvm
bash install.sh

cat >> ~/.bashrc <<'EOF'
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"
export NVM_NODEJS_ORG_MIRROR=https://npmmirror.com/mirrors/node/
EOF

source ~/.bashrc
nvm install 24
nvm use 24
npm config set registry https://registry.npmmirror.com
node -v
npm -v
macOS zsh 如果你用 zsh，复制这段追加配置 
cat >> ~/.zshrc <<'EOF'
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"
export NVM_NODEJS_ORG_MIRROR=https://npmmirror.com/mirrors/node/
EOF

source ~/.zshrc
nvm install 24
nvm use 24
Windows 先装 Git，再装 Node.js 24
下载 Git 64-bit 下载 Node.js 24 x64

安装完成后重新打开 PowerShell，再执行下面这段。如果 PowerShell 提示脚本执行被禁止，先复制下面这条命令。

Windows 执行策略 PowerShell 
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Windows npm 镜像 PowerShell 
npm config set registry https://registry.npmmirror.com
node -v
npm -v
git --version

2. 安装 AI CLI

Node.js 24 和 npm 镜像配置好之后，复制下面命令安装。

Claude Code npm 安装 
npm i -g @anthropic-ai/claude-code
其他安装方式 
# Linux / macOS / WSL
curl -fsSL https://claude.ai/install.sh | bash

# Windows PowerShell
irm https://claude.ai/install.ps1 | iex
Codex npm 安装 
npm i -g @openai/codex
OpenClaw 安装和配置 
npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw config
openclaw --version

3. 配置样例

Claude Code 和 Codex 的常用配置示例。Codex 这里拆成 `auth.json` 和 `config.toml` 两个文件。

Claude Code settings.json 放到 ~/.claude 
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "sk-your-token",
    "ANTHROPIC_BASE_URL": "https://awesome-token.com"
  },
  "permissions": {
    "defaultMode": "bypassPermissions"
  },
  "enabledPlugins": {
    "document-skills@anthropic-agent-skills": true,
    "superpowers@claude-plugins-official": false
  },
  "extraKnownMarketplaces": {
    "karpathy-skills": {
      "source": {
        "source": "github",
        "repo": "forrestchang/andrej-karpathy-skills"
      }
    }
  },
  "skipWebFetchPreflight": true,
  "autoCompactEnabled": true,
  "skipDangerousModePermissionPrompt": true,
  "alwaysThinkingEnabled": true
}
Codex auth.json 和 config.toml
auth.json 放到 ~/.codex 
{
  "OPENAI_API_KEY": "sk-****"
}
config.toml 放到 ~/.codex 
personality = "pragmatic"
model = "gpt-5.5"
model_reasoning_effort = "high"
model_provider = "awe"
approvals_reviewer = "user"
approval_policy = "never"
sandbox_mode = "danger-full-access"

[model_providers.awe]
name = "awe"
base_url = "https://awesome-token.com/v1"
wire_api = "responses"
requires_openai_auth = true

[notice]
hide_full_access_warning = true
